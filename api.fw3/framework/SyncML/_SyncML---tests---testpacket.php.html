Script to test individual SyncML messages.

The scripts takes a single client message, either XML or WBXML encoded, and tries to parse it and generate a response message. It doesn't talk to any backend, so it's not able to test the actualy command being sent in the message. Its purpose is to make sure that SyncML messages are correctly and completely parsed and distributed into the business logic.
