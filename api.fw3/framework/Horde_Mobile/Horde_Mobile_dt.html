This class defines the terms of a Horde_Mobile_dl object.

Examples:

$image1 = new Horde_Mobile_image("my_image.wbmp", "my_image.png", ":-)"); $text1 = new Horde_Mobile_text("my text"); $dt1 = new Horde_Mobile_dt(); $dt1->add($image1); $dt1->add(); $dt1->add($text1);
