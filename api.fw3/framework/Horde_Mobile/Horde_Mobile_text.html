Horde_Mobile_text

Description
Description | Vars (details) | Methods (details)

This class inserts plain text into a Horde_Mobile_block or a Horde_Mobile_row object.

Examples:

$myText1 = new Horde_Mobile_text("Hello World"); $myText2 = new Horde_Mobile_text("Welcome to Horde_Mobile", 'b'); $myText3 = new Horde_Mobile_text("Good Morning", array('b', 'big'));
