Horde_Mobile_block

Description
Description | Descendents | Vars (details) | Methods (details)

This class holds text-level elements for use in Horde_Mobile or Horde_Mobile_form objects.

Examples:

$block = new Horde_Mobile_block("Hello World"); $text = new Horde_Mobile_text("Welcome to Horde_Mobile", 'b'); $block->add($text);
