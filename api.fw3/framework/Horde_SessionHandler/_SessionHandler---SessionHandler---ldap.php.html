SessionHandler implementation for LDAP directories.

Required parameters:

   'hostspec' - (string) The hostname of the ldap server.
   'port'     - (integer) The port number of the ldap server.
   'dn'       - (string) The bind DN.
   'password' - (string) The bind password.
