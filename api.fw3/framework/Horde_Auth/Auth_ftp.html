Auth_ftp

Description
Description | Vars | Methods (details)

The Auth_ftp class provides an FTP implementation of the Horde authentication system.

Optional parameters:

   'hostspec'  The hostname or IP address of the FTP server.
               DEFAULT: 'localhost'
   'port'      The server port to connect to.
               DEFAULT: 21
