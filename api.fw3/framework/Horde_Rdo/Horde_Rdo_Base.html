		getMapper (line 240)
	
 
	
	

Get a Mapper instance that can be used to manage this  object. The Mapper instance can come from a few places:


  • If the class <RdoClassName>Mapper exists, it will be used
   automatically.
 
  • Any Rdo instance created with Horde_Rdo_Mapper::map() will have a
   $mapper object set automatically.
 
  • Subclasses can override getMapper() to return the correct
   mapper object.
 
  • The programmer can call $rdoObject->setMapper($mapper) to provide a
   mapper object.
  A Horde_Rdo_Exception will be thrown if none of these  conditions are met.

	
    
				
  • return: The Mapper instance managing this object.
    • 
				
  • access: public
