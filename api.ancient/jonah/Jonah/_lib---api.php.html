_jonah_searchTags (line 297)

Searches internal channels for stories tagged with all requested tags.

Returns an application-agnostic array (useful for when doing a tag search across multiple applications) containing the following keys: 

  'title'    - The title for this resource.
  'desc'     - A terse description of this resource.
  'view_url' - The URL to view this resource.
  'app'      - The Horde application this resource belongs to.

The 'raw' story array can be returned instead by setting $raw = true.

  • return: An array of results | PEAR_Error
mixed _jonah_searchTags (array $names, [integer $max = 10], [integer $from = 0], [array $channel_id = array()], [integer $order = 0], [boolean $raw = false])
  • array $names: An array of tag_names to search for (AND'd together).
  • integer $max: The maximum number of stories to return.
  • integer $from: The number of the story to start with.
  • array $channel_id: An array of channel_ids to limit the search to.
  • integer $order: How to order the results (a JONAH_ORDER_* constant)
  • boolean $raw: Return the raw story data?
