What is Midas?
--------------

Midas is the Horde module for serving and managing ads/banners. It is
still in the very early stages of development.

This software is OSI Certified Open Source Software.
OSI Certified is a certification mark of the Open Source Initiative.


Obtaining Midas
---------------

Further information on Midas and the latest version can be obtained
at 

  http://www.horde.org/midas/
