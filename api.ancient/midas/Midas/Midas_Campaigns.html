		singleton (line 590)
	
 
	
	

Attempts to return a reference to a concrete Midas_Driver instance based  on $driver.


It will only create a new instance if no Midas_Driver instance with the  same parameters currently exists.
This should be used if multiple storage sources are required.
This method must be invoked as: $var = &Midas_Driver::singleton()

	
    
				
  • return: The created concrete Midas_Driver instance, or false                   on error.
