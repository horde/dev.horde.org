Turba_Driver_Facebook

Description
Description | Vars | Methods (details)

Read-only Turba directory driver for Facebook friends. Requires a Horde application to be setup on Facebook and configured in horde/config/conf.php.

This driver based on the favourites driver.

Of limited utility since email addresses are not retrievable via the Facebook API, unless the user allows the Horde application to access it - and even then, it's a proxied email address.
