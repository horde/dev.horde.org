		singleton (line 76)
	
 
	
	

Attempts to return a reference to a concrete Nic_Storage instance  based on $driver.  It will only create a new instance if no  Nic_Storage instance with the same parameters currently exists.


This should be used if multiple storage sources are required.
This method must be invoked as: $var = &Nic_Storage::singleton()
