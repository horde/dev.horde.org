		singleton (line 129)
	
 
	
	

Attempts to return a reference to a concrete Nic_Device instance based  on $driver.


It will only create a new instance if no Nic_Device instance with the  same parameters currently exists.
This should be used if multiple storage sources are required.
This method must be invoked as: $var = &Nic_Device::singleton()

	
    
				
  • return: The created concrete Nic_Device instance, or false                   on error.
