SyncML_Command_Results

Description
Description | Vars (details) | Methods

The SyncML_Command_Results class provides a SyncML implementation of the Results command as defined in SyncML Representation Protocol, version 1.1, section 5.5.12.

The Results command is used to return the results of a Search or Get command. Currently SyncML_Command_Results behaves the same as SyncML_Command_Put. The only results we get is the same DevInf as for the Put command.
