Prefs_UI

Description
Description | Vars (details) | Methods (details)

Class for auto-generating the preferences user interface and processing the forms.

Set $_SESSION['horde_prefs']['nomenu'] to true to suppress output of the Horde_Menu on the options pages.

For 'special' group types, set 'prefsui_no_save' to suppress printing of the "Save Changes" and "Undo Changes" buttons.
