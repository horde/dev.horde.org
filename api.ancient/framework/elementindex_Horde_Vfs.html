Horde_Vfs_Adapter_Db_Base
Horde_Vfs_Adapter_Db_Base in Base.php
This Db base class manages the horde_vfs_metadata table for the various Db backends (Chunked, File, S3). The metadata table holds the filesystem representation in the database, but actual file contents are not stored thre. A subclass is needed to provide actual file storage.

A file-backed Vfs adapter that uses a database for the filesystem tree and metadata, and stores file contents in chunks in a separate blob table. Large files can be split between many chunks; small files just take one.
