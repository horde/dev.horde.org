Horde_Mobile_dl

Description
Description | Vars (details) | Methods (details)

This class allows to insert definition lists into a Horde_Mobile or Horde_Mobile_form object.

Examples:

$myDl = new Horde_Mobile_dl();

$dt1 = new Horde_Mobile_dt(); $dt1->add($image1); $dt1->add($text1); $myDl->add($dt1);

$dt2 = new Horde_Mobile_dt(); $dt2->add($image2); $dt2->add($text2); $myDl->add($dt2);

$myDeck->add($myDl);
