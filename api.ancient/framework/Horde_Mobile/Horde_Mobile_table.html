Horde_Mobile_table

Description
Description | Vars (details) | Methods (details)

This class allows to insert tables into a Horde_Mobile or Horde_Mobile_form object.

Examples:

$myTable = new Horde_Mobile_table();

$row1 = new Horde_Mobile_row(); $row1->add($image1); $row1->add($text1); $myTable->add($row1);

$row2 = new Horde_Mobile_row(); $row2->add($image2); $row2->add($text2); $myTable->add($row2);

$myDeck->add($myTable);
