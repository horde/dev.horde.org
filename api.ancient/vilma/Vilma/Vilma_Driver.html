		singleton (line 399)
	
 
	
	

Attempts to return a reference to a concrete Vilma_Driver instance  based on $driver.


It will only create a new instance if no Vilma_Driver instance with the  same parameters currently exists.
This should be used if multiple storage sources are required.
This method must be invoked as: $var = &Vilma_Driver::singleton()

	
    
				
  • return: The created concrete Vilma_Driver instance, or false on                 error.
