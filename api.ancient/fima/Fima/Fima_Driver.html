		singleton (line 397)
	
 
	
	

Attempts to return a reference to a concrete Fima_Driver  instance based on $driver. It will only create a new instance  if no Fima_Driver instance with the same parameters currently  exists.


This should be used if multiple storage sources are required.
This method must be invoked as: $var = &Fima_Driver::singleton()

	
    
				
  • return: The created concrete Fima_Driver instance, or false                   on error.
    • 
			

	
	

		mixed
		
			&singleton
		
					([string $ledger = ''], [string $driver = null], [array $params = null])
			

	
			
    
					
  • 
				string
				$ledger: The name of the ledger to load.			
    • 
					
  • 
				string
				$driver: The type of concrete Fima_Driver subclass                               to return.  The is based on the storage                               driver ($driver).  The code is dynamically                               included.			
    • 
					
  • 
				array
