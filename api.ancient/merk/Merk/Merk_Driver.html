Attempts to return a reference to a concrete Merk_Driver instance based  on $driver.


It will only create a new instance if no Merk_Driver instance with the  same parameters currently exists.
This should be used if multiple storage sources are required.
This method must be invoked as: $var = &Merk_Driver::singleton()
