		singleton (line 162)
	
 
	
	

Attempts to return a reference to a concrete Ulaform_Driver instance  based on $driver.


It will only create a new instance if no Ulaform_Driver instance with  the same parameters currently exists.
This should be used if multiple storage sources are required.
This method must be invoked as: $var = &Ulaform_Driver::singleton()
