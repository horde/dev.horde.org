			array
			$_attributes
			 = array()			(line 49)
		
	


	

Hash containing this Horde Policy attributes.


Each attribute is stored as a multi-dimensional array with the following  structure:
$_attribute['app']['property']['value']
where:
'app' is one of the apps found in the registry.
'property' is the name of one of the attribute properties:       'name', 'value', 'overridable', 'type', 'description'
'value' is the value of the property
