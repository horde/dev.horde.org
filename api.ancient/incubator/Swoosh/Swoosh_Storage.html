		singleton (line 128)
	
 
	
	

Attempts to return a reference to a concrete Swoosh_Storage instance  based on $driver.


It will only create a new instance if no Swoosh_Storage instance with  the same parameters currently exists.
This should be used if multiple storage sources are required.
This method must be invoked as: $var = &Swoosh_Storage::singleton()

	
    
				
  • return: The created concrete Swoosh_Storage instance, or false on                 error.
