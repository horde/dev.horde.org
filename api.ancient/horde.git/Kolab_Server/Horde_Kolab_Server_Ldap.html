read (line 141)

Low level access to reading object data.

This function provides fast access to the Server data.

Usually you should use

$object = $server->fetch('a server uid'); $variable = $object['attribute']

to access object attributes. This is slower but takes special object handling into account (e.g. custom attribute parsing).

  • return: An array of attributes or false if the specified object was not found.
  • throws: Horde_Kolab_Server_Exception If the search operation retrieved a problematic result.
  • access: public
