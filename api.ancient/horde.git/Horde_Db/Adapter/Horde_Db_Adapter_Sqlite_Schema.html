Remove the given index from the table.

Remove the suppliers_name_index in the suppliers table (legacy support, use the second or third forms). remove_index :suppliers, :name Remove the index named accounts_branch_id in the accounts table. remove_index :accounts, :column => :branch_id Remove the index named by_branch_party in the accounts table. remove_index :accounts, :name => :by_branch_party
