connect (line 108)

Connect to the db

MySQLi can connect using SSL if $config contains an 'ssl' sub-array containing the following keys:

  • key The path to the key file.
  • cert The path to the certificate file.
  • ca The path to the certificate authority file.
  • capath The path to a directory that contains trusted SSL CA certificates in pem format.
  • cipher The list of allowable ciphers for SSL encryption.
Example of how to connect using SSL:
  1.  $config = array(
  2.      'username' => 'someuser',
  3.      'password' => 'apasswd',
  4.      'hostspec' => 'localhost',
  5.      'database' => 'thedb',
  6.      'ssl'      => array(
  7.          'key'      => 'client-key.pem',
  8.          'cert'     => 'client-cert.pem',
  9.          'ca'       => 'cacert.pem',
  10.          'capath'   => '/path/to/ca/dir',
  11.          'cipher'   => 'AES',
  12.      ),
  13.  );
  14.  
  15.  $db = new Horde_Db_Adapter_Mysqli($config);
