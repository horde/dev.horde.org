		singleton (line 120)
	
 
	
	

Attempts to return a reference to a concrete Skoli_Driver  instance based on $driver. It will only create a new instance  if no Skoli_Driver instance with the same parameters currently  exists.


This should be used if multiple storage sources are required.
This method must be invoked as: $var = &Skoli_Driver::singleton()

	
    
				
  • return: The created concrete Skoli_Driver instance, or false                   on error.
