		singleton (line 407)
	
 
	
	

Attempts to return a reference to a concrete Genie_Driver instance based  on $driver.


It will only create a new instance if no Genie_Driver instance with the  same parameters currently exists.
This should be used if multiple storage sources are required.
This method must be invoked as: $var = &Genie_Driver::singleton()

	
    
				
  • return: The created concrete Genie_Driver instance, or false                   on error.
