Parameters

$text
This is the full text of the MIME message.
$info
<p>An array of information for the new ticket. This should include: - 'queue' => queue id - 'type' => type id - 'state' => state id - 'priority' => priority id - 'ticket' => ticket id (prevents creation of new tickets)</p>
$auth_user
<p>This will be the Horde user that creates the ticket. If null, we will try to deduce from the message's From: header. We do NOT default to $GLOBALS['registry']->getAuth().</p>
