Class Horde_SyncMl_Command_Final

The Horde_SyncMl_Command_Final class provides a SyncML implementation of the Final command as defined in SyncML Representation Protocol, version 1.1, section 5.1.7.

The Final command is an indicator that the SyncML message is the last message in the current Horde_SyncMl package.
