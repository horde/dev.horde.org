Class Horde_Mime_Address

The Horde_Mime_Address:: class provides methods for dealing with email address standards (RFC 822/2822/3490/5322).

public static string
# addrObject2String( array $ob, array $opts = array() )

Takes an address object array and formats it as a string.

Takes an address object array and formats it as a string.

Object array format for the address "John Doe john_doe@example.com" is: - host: The host the mailbox is on ("example.com") - mailbox: The user's mailbox ("john_doe") - personal: Personal name ("John Doe")

Parameters

$ob
The address object to be turned into a string.
$opts
<p>Additional options: - charset: (string) The local charset. DEFAULT: NONE - filter: (mixed) A user@example.com style bare address to ignore. Either single string or an array of strings. If the address matches $filter, an empty string will be returned. DEFAULT: No filter - idn: (boolean) Convert IDN domain names (Punycode/RFC 3490) into the local charset. Requires the idn or intl PHP module. DEFAULT: true</p>

Returns

string
The formatted address.
